# Структура аккаунта и рекламных кампаний

Источник: https://study.vincora.ru/tdocs/struktura-akkaunta-i-rk-z1jmi79pb5i5ll2

## 1. ИЕРАРХИЯ СТРУКТУРЫ
**Аккаунт → Рекламная кампания → Группа объявлений → Объявление**

## 2. БЮДЖЕТНЫЕ ТРЕБОВАНИЯ
- На одну РК необходимо 10-15 конверсий еженедельно
- При средней стоимости заявки 800 руб.: минимум 8000 руб/неделю на РК
- Рекомендуемый недельный бюджет: 10000 руб
- Месячный минимум: 40000+ руб

## 3. ПРИНЦИП ВЫБОРА ТИПОВ РК
Приоритизация по типам запросов:
1. **Горячие запросы** (приоритет первый)
2. **Общие запросы** (приоритет второй)
3. **Информационные запросы** (приоритет третий)

При недостатке горячих запросов допускается смешивание общих и информационных в одну группу.

## 4. ОСНОВНОЙ ПРИНЦИП РК
"1 рекламная кампания = 1 направление / товар / услуга / стадия теплоты / вариация поиска"

Внутри каждой РК объявления должны четко описывать конкретный сегмент.

## 5. СТРУКТУРА РК НА ПОИСКЕ (Яндекс Директ)

### Организация групп:
- Группа = Сегмент аудитории / особая характеристика
- До 15 групп в одной РК
- Первый заголовок описывает сегмент
- Второй заголовок и текст тестируются

### Пример структуры (Банкротство):

**РК 1: Банкротство общие**
- Группа 1: Банкротство МФЦ
  - Объявление 1: "Банкротство через МФЦ быстро и с гарантией"
  - Объявление 2: "Банкротство физ лица МФЦ. По договору"

- Группа 2: Бесплатное банкротство
  - Объявление 1: "Бесплатное банкротство в Москве, с гарантией"
  - Объявление 2: "Банкротство бесплатно и под ключ"

**РК 2: Списание долгов**
- Группа 1: Как списать
- Группа 2: Списание под ключ

## 6. СТРУКТУРА РК В РСЯ (Рекламная сеть Яндекса)

### Первая РК: All_categories
**Группы:**
- Группы всех сегментов (ключи + автотаргетинг)
- Группа общих ключей
- Группа ретаргет (посетители сайта)
- Опционально: интересы конкурентов/LAL

**Требования к объявлениям:**
- 5 объявлений с разными заголовками и картинками
- В каждой группе - уникальные объявления под сегмент
- В общих группах - универсальные объявления

### Вторая РК: All_automatic
**Группы:**
- 3 группы с автотаргетингом без ключей
- LAL похожих на конвертеров
- LAL похожих на посетителей

**Объявления:**
- Идентичный набор из 5 объявлений для всех групп

### Третья РК: Test Adv
**Группы (любое количество):**
Каждая группа = 1 критерий ЦА/потребность с одним заголовком и 3 разными картинками

**Условия показа:**
- Широкие ключевые слова ниши
- Автотаргетинг
- Сайты конкурентов
- Аффинити-интересы (200+)
- LAL-аудитории

**Структура объявлений:**
- Каждая группа: 1 уникальный заголовок + 3 разные картинки
- Картинки повторяются в других группах
- Отличие групп только в заголовках

## 7. ПРИНЦИП ГРУППИРОВКИ ОБЪЯВЛЕНИЙ
- Близкие по тематике ключи в одной группе
- Объявления описывают конкретный сегмент
- Иерархия ключей: избегать вложенности (исключить "банкротство в москве 2024" если есть "банкротство в москве")

## 8. ПРИМЕР ПОЛНОЙ СТРУКТУРЫ АККАУНТА (Контекстная реклама)

**Горячие запросы:**
- РК 1: Яндекс директ (горячие) → Заказать/купить | Под ключ
- РК 2: Настройка контекстной рекламы → Заказать/купить | Под ключ

**Общие запросы:**
- РК 3: Яндекс директ → Для... | Москва
- РК 4: Контекстная реклама → Для... | Москва

**Информационные:**
- РК 5: Инфо → Почему/Что выбрать
